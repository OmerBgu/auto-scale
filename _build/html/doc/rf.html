rf module¶

random forest prediction¶

what this model predict?¶

the examined feature are the pod’s cpu, memory and the predicted target us the current network delay at the client that normalized to the formula of TRUNC(IF(H138<1.2,1,IF(H138<1.5,2,3)))

rf.predict(qoe, podsnumber)[source]¶

this function get as parameter the pod’s number and the desired quality of experience it has hard coded path to train csv file with cpu pod1, memory pod1,cpu pod2…. and the score of delay it has hard coded the path for the last 5 seconds of CPU && memory it return 1 if the predicted qoe is greater then the desired qoe means scale up is needed return -1 if the predicted qoe is smaller then the desired qoe means scale up is needed return 0 if the predicted qoe is equal to the desired qoe
